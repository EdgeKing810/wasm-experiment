### Running this demo locally

```bash
git clone https://github.com/EdgeKing810/wasm-experiment.git

cd wasm-experiment/
curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh
wasm-pack build

cd www/
npm i
npm start
```
